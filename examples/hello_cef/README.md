This is an experiment and not ready for any kind of serious use! Keep away or pitch in ;)

A script outside cargo is needed to package the application in the required
structure.

Linux:
```
./linux_build.sh
./linux_run.sh
```

macOS:
```
./macos_build.sh
./macos_run.sh
```

Windows:
```
./windows_build.bat
./windows_run.bat
```
